\Prog{Immio} - A CLI frontend for imm.io

\end{Name}

\section{Synopsis}
\Prog{immio}
\oOpt{-h}
\oOpt{-v}
\oOpt{-d}
\Arg{filename(s)}

\section{Description}
\Prog{Immio} is a very lightweight image uploader for imm.io using \Cmd{curl}{1}.

\section{Options}
\begin{description}
   \item[\Opt{-h}]
      Print help and exit
   \item[\Opt{-v}]
      Provide more verbose information about the uploaded image(s)
   \item[\Opt{-d}]
      Return direct links instead of landing pages
   \item[\Arg{filename(s)}]
      File or files to be uploaded
\end{description}
